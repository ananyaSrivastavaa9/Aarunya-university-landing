# Aarunya University — Modern UI/UX Landing Page (Next.js + Tailwind)

A modern SaaS-style redesign of a university landing page focused on **UI/UX, responsiveness, and clean component architecture**.  
Built using **Next.js (App Router)** and **Tailwind CSS**, designed to demonstrate frontend development skills for internship submission.

---

## Live Preview

👉 https://aarunya-university-landing.vercel.app/

---

## Features

- Modern sticky glassmorphism navbar
- Premium hero section with strong CTA buttons
- About section with feature cards
- Programs/features grid with hover effects
- Student testimonials section
- High-contrast CTA section
- Minimal professional footer
- Fully responsive (mobile, tablet, desktop)
- Smooth scrolling navigation
- Clean reusable component structure
- Modern SaaS-style UI design
